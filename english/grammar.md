## 长难句

### 找长难句主干谓语动词三步骤

#### 看标点 做预判(锁定主干位置)

- [逗号] 非谓语, 主干句, 非谓语. 主干句, 并列句 

- [分号] 主干句; 并列句

  分号 ";" 相当于并列连词 but/and/or 连接的分句

- [冒号/破折号] 主干句: 同位语. 主干句——同位语

  同位语作为名词性短语，起补充解释说明

#### 先数谓语动词，再数连接句子的连词

谓语动词：时间词(be/系)+表状态的非谓语动词——时+态

非谓语动词：主动进行态doing，被动/完成态done，不定态to do

##### 谓语动词3种形式

时态 = 时+态

| 过去时did          | 现在时do            | 将来时will               | 动作状态           |
| ------------------ | ------------------- | ------------------------ | ------------------ |
| **did** do = did   | **do** do = do      | **will do** do = will do | 一般态do           |
| **was/were** doing | **am/is/are** doing | **will be** doing        | 进行态doing        |
| **had** done       | **have** done       | **will have** done       | 完成态done         |
| **had** been doing | **have** been doing | **will have** been doing | 完成进行been doing |

语态

| 过去时did         | 现在时do           | 将来时will              | 动作语态(+时态)         |
| ----------------- | ------------------ | ----------------------- | ------------------------ |
| **was/were** doing | **am/is/are** doing | **will be** doing        | 主动进行doing     |
| **was/were** done | **am/is/are** done | **will be** done        | 被动be done            |
| **had** been doing | **have** been doing | **will have** been doing | 主动+完成进行been doing |
| **had been** done | **have been** done | **will have been** done | 被动+完成have been done |

情态动词+动词原型

can be/do, may be/do, should be/do, must be/do

be able to do, be willing to do, be likely to do, have to do

3种形式的组合

虚拟语态 should have done = should have + have done

have been willing to do = have been willng + be willing to do

### 复杂修饰成分

#### 基本成分

##### 区分did/done同体

中间加be动词，(be) done 被动语态语义成立(不变)时为<过去分词>做定语，否则为谓语 did

a paper (be) published in England. 过去分词做定语

#### 修饰成分

定语、状语

##### 区分that同位语/定语从句

that引导同位语从句不作成分，that引导定语从句指代主语/宾语

##### 区分介宾作定语/状语

根据语义，定语修饰名词

I will hew (out of the mountain (of despair)) a stone of hope

I ate an apple in the kitchen. 介词作状语(可能性大)

I ate the apple in the kitchen. 介词作定语(可能性大)

##### 区分关系副词从句作定语/状语

先行词 + 关系副词引导定语从句

... time/age when ...

... place where ...

... reason/cause why ...

句子 + 关系副词引导状语从句

... when ...

左2 + 名词 + 右6

左2：

- 限定词：冠词，序数词，量词，指代词
- 修饰词：形容词

右6：

- 同位语
- 形容词短语：作后置定语
- 定语从句
- 介词短语
- 不定式
- 现在/过去分词

**分词**

定语从句中(关系代词做主语)简化为非谓语分词：

- <关系代词> be 非谓语 => 非谓语：n. which be + doing/done/to do => n. doing/done/to do
- <关系代词> 谓语 => doing：n. which do/did => n. doing

China is as developing country <u>which belongs</u> to the third world.

China is as developing country <u>belonging</u> to the third world.

Books <u>which are written</u> in English are more expensive.

Books <u>written</u> in English are more expensive.

### 翻译

#### 词意

以语境为主，合理性（引申义），完整性（增/减词）

#### 语序

汉：状语2 状语1 定语1 <u>主语</u> 状语3 <u>谓语</u> 定语2 <u>宾语</u>

英：状语1 状语2            <u>主语</u> 定语1 <u>谓语</u> 状语3 <u>宾语</u> 状语1

英汉基本成分语序一致

英汉修饰成分语序不一致：

- 修饰以倒序译为主
- 并列处顺序不变(分配展开)
- 定语较长时单独译（一般>8个word）
  - 对非限制性定语（逗号分开）加代词：...，(而)这...
  - 按同位语翻译：... 即/也就是/：...

n. of a of b => b 的 a 的 n

I will hew (out of the mountain (of despair)) a stone of hope =>

I will 从 (out of the mountain (of despair)) hew a stone of hope

n. of a and b of c => c 的 (a and b) 的 n. => c 的 a 的 n. and c 的 b 的 n. = c a n. and c b n.
